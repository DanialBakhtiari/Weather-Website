This weather webpage is a collaborative effort between Danial Bakhtiari and Mani Salahshoor. We built it using HTML for structure, CSS for styling, and vanilla JavaScript for interactivity. Tailwind CSS provided a helpful toolkit for rapid development, while Tomorrow.io's API delivers the accurate weather data you see. We leverage Git for version control and utilize GitHub to host and collaborate on the code. We're constantly striving to improve, so feel free to reach out with any feedback or suggestions!
